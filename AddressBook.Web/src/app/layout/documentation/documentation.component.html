AddressBook Application

This is a skill test project. A user can add a contact with an address with coordinates. Afterwards can select any address from his contacts and display it on the map or add another address to any user.


Variations

  1. The address functionality was intented to use an external api to convert address to geolocation, but couln't find any free.
  2. In a real world project repository pattern with Entity Franework is pointless since it is implemented by default. For the purposes of the test was implemented nonetheless.

Libraries

  1. Bulma for ui
  2. Angular for web app
  3. Ngx-openlayers for maps
  4. Ng-snotify as notification messages
